Know what's running

A native macOS menu bar app that shows every TCP port in use on your machine — with deep Docker integration, process management, and developer tooling.

macOS 13 Ventura or later · Free & open source

Features

Everything in the menu bar

No window, no Dock icon — just click the icon and see what's running.

🔍

Smart detection

Automatically recognises dev runtimes — node, python, bun, go, ruby, deno, rust — and highlights them regardless of port number.

⚡

Always up to date

Background polling every 5 seconds. A HUD toast appears the moment a port opens or closes — no menu interaction needed.

🛡️

System service protection

Known macOS system services (AirPlay, Handoff, Sharing) are flagged with an orange shield and require confirmation before killing.

🌐

Open anywhere

One click to open any port in your preferred browser, reveal its folder in Finder, or jump straight into a terminal session.

📊

Resource usage

CPU % and memory shown per process. Docker ports display real container stats from docker stats, not proxy RSS.

🔔

HUD notifications

Native floating toast banners — no system notification permissions required. Works immediately out of the box.

Docker

Deep container integration

Docker ports are resolved to real container names. Grouped by Compose project — one line per stack.

📦

Compose stacks

Containers are grouped by their com.docker.compose.project label — hover a stack to expand all its services.

🎛️

Container controls

Restart, Pause/Unpause, and Stop any container without leaving the menu bar.

📋

Logs & shell

View container logs or open a shell (docker exec -it) directly in your terminal with one click.

🔗

Docker Desktop & OrbStack

Click any container to jump straight to it in Docker Desktop or OrbStack. Auto-detects which one you have installed.

Legend

Colour at a glance

Dev Port — known runtime or configured port number
User Port — other process running under your account
Docker Container (running) — active and serving traffic
Docker Container (paused)
System Service — macOS system process (AirPlay, Handoff…)
Settings

Configurable to your stack

Four-tab settings window — launch at login, preferred browser and terminal, Docker manager, custom port ranges.
